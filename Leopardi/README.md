# L'eco di Leopardi · PWA definitiva

## Contenuto
- lezioni HTML complete
- videoteca
- galleria schemi
- quiz e autoverifica
- glossario con parole-cerniera
- appunti trascinabili con salvataggio `.txt`
- manifest e service worker per uso offline

## Pubblicazione su GitHub Pages
1. Carica tutto il contenuto di questa cartella nella root del repository.
2. Attiva GitHub Pages dal branch principale.
3. Apri `index.html`.

## Nota
Le immagini sono state mantenute nel formato ricevuto. Se in seguito vorrai alleggerire il pacchetto, si può fare una seconda passata dedicata solo all’ottimizzazione.
